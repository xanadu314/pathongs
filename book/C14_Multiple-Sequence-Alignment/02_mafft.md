# Mafft 使用



---

!!! Abstract "内容简介"
    Mafft 是常用的大规模序列比对的工具，在速度和准确度上有很好的平衡。

## 安装

```bash
# conda 安装
(msa)$ conda install mafft
```

## 使用

```bash
# 最基本的序列比对用法
(msa)$ mafft seq.fas > seq.maf
```
