## 基本语法

### 1. 变量赋值

```python
a = 1
b = "hello"
```

### 2. 数据类型

```python
# 字符串
s = "abc"
s = "中文"

# 数字
x = 1
y = 0.01

# 列表
a = [1, 2, 3]

# 元组
b = (1, 2, 3)

# 字典
c = {'var1': 100, 'var2': 200}
```


### 3. 导入模块

```python
import Bio
from Bio import SeqIO
```

### 4. 循环控制

```python
for i in range(0, 100):
    ...

while True:
    ...
```

### 5. 条件判断

```python
if a == 100 and b is True or c:
    print("wrong")

if a <100:
    print("abc")
elif:
    print("def")
else:
    print("ghi")
```

### 6. 函数定义与调用

```python
def my_func(*args, **kwargs):
    for k, v in kwargs:
        return k, v

# 运行函数
print("hello world")

from math import sqrt
sqrt(2)
```
